# Projeto Final de JavaScript

## Enunciado

Boas notícias! Você acaba de ser contratado como desenvolvedor front-end júnior pelo Cesae Digital!

Estamos muito animados em tê-lo na equipe, mas precisamos da sua ajuda para dar um "tapa" no nosso site. Ele tem um grande potencial, mas precisa de alguns ajustes para ficar mais moderno, interativo e funcional.

Seu primeiro desafio é analisar o código atual e corrigir alguns erros que estão impedindo o site de funcionar como esperado. Além disso, queremos que você use suas habilidades em JavaScript para implementar novas funcionalidades e melhorar a experiência dos nossos usuários.

Aqui está a lista de tarefas que preparamos para você:

### Missão 1: Correção de Bugs

- Parece que o formulário de contacto não está funcionando como deveria. Investigue o código JavaScript e corrija os erros para que ele possa ser enviado corretamente.
- A animação dos cards dos cursos também estão com problemas. Dê uma olhada no código e faça os ajustes necessários para que a animação funcione sem falhas.
- Nós estamos crescendo e no nosso texto sobre nós queremos incluir cursos em mais duas áreas em nossa lista (Alteração deve ser feita apenas através do ficheiro de JavaScript):
  - Comunicação Digital;
  - Formação de formadores

### Missão 2: De Volta ao Topo

- Adicione um botão "Voltar ao Topo" que leve o usuário ao início da página quando clicado.
- Esse botão deve ser fixo no canto inferior direito da tela e só aparecer quando o usuário rolar a página para baixo.

### Missão 3: Formulário de Contacto Turbinado

- O formulário de contacto precisa de uma validação mais completa. Valide todos os campos (nome, email, contacto, região, mensagem) e exiba mensagens de erro específicas para cada campo inválido.
- Para melhorar a experiência do usuário, o formulário não deve ser enviado se houver campos inválidos.

### Missão 4: Desafios Extra

- **Ninja do AJAX**: Implemente o envio do formulário de contcto sem recarregar a página, usando AJAX. Exiba uma mensagem de sucesso após o envio.
- **Mestre dos Modais**: Crie modais (janelas pop-up) para cada curso, exibindo mais informações sobre eles (ex: info, duração, valor). Os modais devem ser abertos ao clicar no botão "Saiba Mais" do card correspondente.
- **Rei do Quiz**: Crie um quiz interativo com perguntas e respostas sobre o Cesae Digital ou sobre os temas do curso. Exiba as perguntas, valide as respostas e mostre o resultado final.
- **Lorde do Carrossel**: Implemente um carrossel de imagens na secção "Sobre Nós", exibindo diferentes imagens relacionadas ao CESAE Digital
- **Imperador das APIs**: Integre a página com APIs externas, como a API do Google Maps para mostrar a localização do Cesae Digital.
