Sobre este trabalho

Website desenvolvido no âmbito do módulo Programação para a Web - cliente (client-side), por Ana Carolina Pinto.

O design do website é original, no entanto as receitas, as imagens e as descrições foram retiradas do blog Beyond the Bayou.

O vídeo da página inical é da criadora de conteúdo Turkuaz Kitchen, tendo sido retirado o som original e acrescentado uma música livre de direitos autorais.

Os ícones (os cãezinhos) são da autoria do artista Yowoyowoo.

O idioma da página é o inglês, pois o blog de onde o conteúdo foi retirado também está escrito em inglês.

Elementos não pedidos incluídos nas páginas
  • Arcordião (nas páginas das receitas)
  • Formulário de login
  • Formulário de subscrição de Newsletter, com checkbox personalizada

Os formulários do website não estão funcionais, pois não foi feita uma integração com uma base de dados. Como tal a informação neles colocada não será armazenada.

Auto-avaliação

Todos os elementos pedidos no ponto 1 e 2 do enunciado do trabalho foram incluídos no website. Penso também que o código e o projeto estejam bem estruturados e compreensíveis.

Assim sendo, a minha auto-avaliação para este projeto é 19 valores.
